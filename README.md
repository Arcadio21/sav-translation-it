## Scum And Villainy - Italiano
Modulo di traduzione per Scum And Villainy  System per Foundry VTT.
Istruzioni
installa il sistema di Scum And Villainy 
installa Babele
installa questo modulo
attiva i moduli nel pannello di controllo
cambia lingua in italiano nel pannello di controllo.
